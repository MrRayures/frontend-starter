Breakpoints

Pour ajouter un breakpoint Ã  une classe utilitaire il suffit d'y ajouter le suffix suivant @{bp} ce qui nous donne : u-ma-classe-utilitaire@{bp}.

Breakpoints disponible :

Remplacer {bp} par les breakpoints suivants :

    {% for bp_name, bp_value in css_breakpoints %}
  • {{bp_name}} : {{bp_value}} ({{(bp_value | int)*16}}px)
    • {% endfor %}

Exemple avec box-shadow :

u-box-shadow-lg
u-box-shadow-md@md
u-box-shadow-sm@sm
{% markdown %} ```html
u-box-shadow-lg
u-box-shadow-md@md
u-box-shadow-sm@sm
``` {% endmarkdown %}

Utilisation en scss :

Pour utiliser les breakpoints en scss il suffit d'utiliser la fonction suivante : bp({bp}) avec le mixins mq().
En savoir plus sur les mixins.

{% markdown %} ```css .component { border-color: color(primary); @include mq(bp(md)){ border-color: color(secondary); } @include mq(bp(sm)){
border-color: color(success); } } ``` {% endmarkdown %}

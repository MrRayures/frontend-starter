Radius

Pour ajouter un radius à un élement il suffit d'y ajouter la classe suivante : u-radius-{size}.

Tailles disponibles :

Remplacer {size} par les valeurs suivantes :

    {% for radius_name, radius_val in css_radius %}
  • {{ radius_name}} : {{ radius_val}}
    • {% endfor %}

Exemple :

{% for radius_name, radius_val in css_radius %}
Hello World ({{ radius_val}})
{% endfor %}
{% markdown %} ```html {% for radius_name, radius_val in css_radius %}
Hello World ({{ radius_val}})
{% endfor %} ``` {% endmarkdown %}

Utilisation en scss :

Pour ajouter un radius à un élement en scss il suffit d'utiliser la fonction suivante : radius({size}).

{% markdown %} ```css .component { border-radius: radius(md); } ``` {% endmarkdown %}

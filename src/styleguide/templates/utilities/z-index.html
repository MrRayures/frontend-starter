Utilisation en scss :

Pour ajouter un z-index à un élement en scss il suffit d'utiliser la fonction suivante : zIndex({index}).
Remplacer {index} par les valeurs disponible ci-dessus.

Exemple :

{% markdown %} ```css .component { z-index: zIndex(overlay); } ``` {% endmarkdown %}

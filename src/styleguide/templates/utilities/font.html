Font

Les fonts disponibles sont référencées dans le guide des typographies

font-family

Pour utiliser une font-family sur un élement il suffit d'y ajouter la classe suivante : u-font-{font}.

Font disponibles :

Remplacer {font} par les valeurs suivantes :

    {% for font_name, font_val in css_font_families %}
  • {{ font_name }} = {{ font_val}}
    • {% endfor %}

Exemple :

{% for font_name, font_val in css_font_families %}

Hello {{ font_val}}

{% endfor %}
{% markdown %} ```html {% for font_name, font_val in css_font_families %}

Hello {{ font_val}}

{% endfor %} ``` {% endmarkdown %}

font-size

Pour régler la font-size sur un élement il suffit d'y ajouter la classe suivante : u-font-{size}.
Remplacer {size} Les valeurs disponibles :

    {% for font_size_name, font_size_val in css_font_sizes %}
  • {{ font_size_name}} = {{ font_size_val}}
    • {% endfor %}

Exemple :

{% for font_size_name, font_size_val in css_font_sizes %}

Hello lorem ipsum ({{ font_size_val}})

{% endfor %}
{% markdown %} ```html {% for font_size_name, font_size_val in css_font_sizes %}

Hello lorem ipsum ({{ font_size_val}})

{% endfor %} ``` {% endmarkdown %}

font-weight

Pour régler la font-size sur un élement il suffit d'y ajouter la classe suivante : u-font-{weight}.
Remplacer {weight} Les valeurs disponibles :

    {% for font_weight_name, font_weight_val in css_font_weight %}
  • {{ font_weight_name}} = {{ font_weight_val}}
    • {% endfor %}
{% for font_weight_name, font_weight_val in css_font_weight %}

Hello lorem ipsum ({{ font_weight_val}})

{% endfor %}
{% markdown %} ```html {% for font_weight_name, font_weight_val in css_font_weight %}

Hello lorem ipsum ({{ font_weight_val}})

{% endfor %} ``` {% endmarkdown %}

Utilisation en scss :

Pour ajouter une font-family, font-size et une font-weight à un élement en scss il suffit d'utiliser la fonction suivante : font({font}).

{% markdown %} ```css .component { font-family: font(primary); } .component { font-size: fontSize(12); } .component { font-weight: fontWeight(400); } ``` {% endmarkdown %}

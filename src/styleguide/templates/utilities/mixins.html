Mixins

A utiliser en scss.

@include mq()

A quoi ça sert ?

Gestion du responsive via @media en css.

Comment l'utiliser ?

@include mq(MAX, MIN) {...}

  • @include mq(50rem) = @media (max-width: 50rem)
  • @include mq(0, 50rem) = @media (min-width: 50rem)
  • @include mq(50rem, 32rem) = @media (min-width: 32rem) and (max-width: 50rem)

Exemple

{% markdown %} ```css .component { max-width: 5rem; @include mq(50rem){ max-width: 2rem; } } ``` {% endmarkdown %}

L'utiliser avec la fonction bp({size})

par exemple : @include mq(bp(sm)) = @media (max-width: 768px)
Liste des breakpoints disponbiles.

{% markdown %} ```css .component { max-width: 5rem; @include mq(bp(sm)) { max-width: 2rem; } } ``` {% endmarkdown %}

@include ie()

Pour ajouter une propriété css juste pour IE11 & IE10 il suffit d'utiliser le mixin suivant : ie().

{% markdown %} ```css .component { box-shadow: shadow(sm); @include ie(){ box-shadow: none; border-bottom:1px solid #cccccc; } } ``` {% endmarkdown %}

@include edge()

Pour ajouter une propriété css juste pour Edge il suffit d'utiliser le mixin suivant : edge().

{% markdown %} ```css .component { box-shadow: shadow(sm); @include edge(){ box-shadow: none; border-bottom:1px solid #cccccc; } } ``` {% endmarkdown %}

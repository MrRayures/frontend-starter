Grille

Par défaut, les colonnes occupent tout l'espace disponible avec une largeur égale.

1
2
{% markdown %} ```html
1
2
``` {% endmarkdown %}

Si vous devez définir une largeur, vous pouvez utiliser des modificateurs o-col basés sur les numéros de colonne.
Exemple : o-col--8 & o-col--4

8
4
{% markdown %} ```html
8
4
``` {% endmarkdown %}

Breakpoints

Plus d'infos sur les breakpoints.

812
412
{% markdown %} ```html
812
412
``` {% endmarkdown %}

Grid offset

Paramétrage des offset avec o-col--offset-{X} où {X} est le nombre de colonne.
Exemple :

8
45
25
812
{% markdown %} ```html
8
45
25
812
``` {% endmarkdown %}

Couleur de texte

Pour ajouter une couleur à un élement il suffit d'y ajouter la classe suivante : u-color-{color}.

Couleurs disponibles :

Remplacer {color} par les couleurs disponibles :

    {% for color in colors %}
  • u-color-{{color.name}}
    • {% endfor %}

Démo :

u-color-primary

Couleur de fond

Pour ajouter une couleur de fond à un élement il suffit d'y ajouter la classe suivante : u-bgcolor-{color}.

Couleurs disponibles :

    {% for bgcolor in bgcolors %}
  • u-color-{{bgcolor.name}}
    • {% endfor %}

Démo :

{% for bgcolor in bgcolors %}
u-bgcolor-{{bgcolor.name}}
{% endfor %}

Utilisation en scss :

Pour ajouter une couleur à un élement en scss il suffit d'utiliser la fonction suivante : color({color}).

.component {
  color: color(primary);
}

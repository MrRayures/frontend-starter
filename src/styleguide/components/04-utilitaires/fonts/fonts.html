font-family

Pour utiliser une font-family sur un élement il suffit d'y ajouter la classe suivante : u-font-{font}.

Font disponibles :

Remplacer {font} par les valeurs suivantes :

    {% for font in fonts %}
  • {{font}} :
    • {% endfor %}

Exemple :

{% for font in fonts %}
Hello
{% endfor %}

font-size

Pour régler la font-size sur un élement il suffit d'y ajouter la classe suivante : u-font-{size}.
Remplacer {size} par les valeurs disponibles :

    {% for size in sizes %}
  • {{size}}
    • {% endfor %}

font-weight

Pour régler la font-size sur un élement il suffit d'y ajouter la classe suivante : u-font-{weight}.
Remplacer {weight} Les valeurs disponibles :

Utilisation en scss :

Pour ajouter une font-family à un élement en scss il suffit d'utiliser la fonction suivante : font({font}).

.component {
  font-family: font(primary);
}

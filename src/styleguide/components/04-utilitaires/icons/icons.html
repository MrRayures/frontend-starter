Width & Height

Pour ajouter une taille à une icône il suffit d'y ajouter la classe suivante : u-icon-{size}.

Tailles disponibles :

Remplacer {size} par les tailles suivantes :

    {% for size in sizes %}
  • {{size}} :
    • {% endfor %}
{% for size in sizes %}
{{size}}
{% endfor %}

Couleurs

Pour ajouter une taille à une icône il suffit d'y ajouter la classe suivante : u-icon-{color}.

Couleurs disponibles :

Remplacer {color} par les couleurs suivantes :

    {% for color in colors %}
  • {{color}}
    • {% endfor %}
{% for color in colors %}
{{color}}
{% endfor %}

Utilisation en scss :

our ajouter une taille à une icône en scss il suffit d'utiliser la fonction suivante : iconSize({size}).

.component {
  width: iconSize(16);
  height: iconSize(16);
}

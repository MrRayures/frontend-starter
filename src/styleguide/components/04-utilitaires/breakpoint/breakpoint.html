Breakpoints

Pour ajouter un breakpoint à une classe utilitaire il suffit d'y ajouter le suffix suivant @{bp} ce qui nous donne : u-ma-classe-utilitaire@{bp}.

Attention, pour des raisons de performance CSS les classes utilitaires responsive sont à activer aux besoins !

Breakpoints disponible :

Remplacer {bp} par les breakpoints suivants :

    {% for bp in bps %}
  • {{bp.name}} :
    • {% endfor %}

Exemple avec du padding :

u-p-20 u-p-0@lg u-p-20@md u-p-0@sm

Utilisation en scss :

Pour utiliser les breakpoints en scss il suffit d'utiliser la fonction suivante : bp({size}) avec le mixins mq().

Mixin @include mq()

A utiliser en scss.

A quoi ça sert ?

Gestion du responsive via @media en css.

Comment l'utiliser ?

@include mq(MAX, MIN) {...}

  • @include mq(50rem) = @media (max-width: 50rem)
  • @include mq(0, 50rem) = @media (min-width: 50rem)
  • @include mq(50rem, 32rem) = @media (min-width: 32rem) and (max-width: 50rem)

Exemple

.component {
  max-width: 5rem;
  @include mq(50rem){
    max-width: 2rem;
  }
}

L'utiliser avec la fonction bp({size})

@include mq(bp(sm)) = @media (max-width: 768px)

.component {
  max-width: 5rem;
  @include mq(bp(sm)){
    max-width: 2rem;
  }
}

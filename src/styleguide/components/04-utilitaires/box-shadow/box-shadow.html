Pour ajouter une box-shadow à un élement il suffit d'y ajouter la classe suivante : u-box-shadow-{size}.

Classes disponibles :

Remplacer {size} par les tailles suivantes :

    {% for boxShadow in boxShadows %}
  • {{boxShadow.name}}
    • {% endfor %}
{% for boxShadow in boxShadows %} u-box-shadow-{{boxShadow.name}} {% endfor %}

Utilisation en scss :

Pour ajouter une box-shadow à un élement en scss il suffit d'utiliser la fonction suivante : shadow({size}).

.component {
  box-shadow: shadow(md);
}

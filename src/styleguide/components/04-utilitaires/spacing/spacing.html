Classes utilitaires responsives disponiblent u-{property}{direction}-{size}@{bp}

Pour ajouter un espace à un élement il suffit d'y ajouter la classe suivante : u-{property}{direction}-{size}.

Margin ou padding

Remplacer {property} par les valeurs suivantes :

  • p pour padding
  • m pour margin

Top, right, bottom ou left

Si {direction} n'est pas définie l'espace s'appliquera dans toutes les directions.
Remplacer {direction} par les valeurs suivantes :

  • t pour top
  • r pour right
  • b pour bottom
  • l pour left
  • y pour left & right
  • x pour top & bottom

Tailles disponibles :

Remplacer {size} par les valeurs disponible :

    {% for spacingVal in spacing %}
  • {{spacingVal.name}} :
    • {% endfor %}

Ce qui donne par exemple : u-mb-20 = margin-bottom: 2rem;

{% for spacingVal in spacing %}
{{spacingVal.name}}
{% endfor %}

Utilisation en scss :

Pour ajouter un espace à un élement en scss il suffit d'utiliser la fonction suivante : spacing({size}).

.component {
  padding-top: spacing(md);
}

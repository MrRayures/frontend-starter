Spacing

Pattern for spacing utility class : u-{property}{sides}-{size}

{property} :
p is for padding
m is for margin

{sides} :
t is for top
r is for right
b is for bottom
l is for left

{size} :
xs, sm, md, lg, xlg, xxlg, xxxlg auto none

Exemple with margin-top :
.u-mt-xs, .u-mt-sm, .u-mt-md, .u-mt-lg, .u-mt-xlg, .u-mt-xxlg, .u-mt-xxxlg .u-mt-auto .u-mt-none
